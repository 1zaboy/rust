Holds a multi-dimensional array of elements of a single data type.

For all types other than strings, the data buffer stores elements in row major order. E.g. if data is treated as a vector of T:

  element 0:   index (0, ..., 0)
  element 1:   index (0, ..., 1)
  ...

The layout for strings is currently undefined.
