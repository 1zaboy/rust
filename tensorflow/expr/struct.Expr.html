pub struct Expr<T: TensorType> { /* fields omitted */ }

A operation in an expression tree, which is a thin wrapper around an ExprImpl.

This is separate from ExprImpl because we want expressions to be wrapped in an Rc, and we can't directly implement std::ops::Add, etc., for Rc>.
